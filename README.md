This is a simple event management webpage created using HTML. The page shows information about upcoming events, including a photo gallery event and a fashion store event. The goal of this project is to practice basic HTML structure and understand how to organize content on a webpage.

## Project Overview

The website displays event details such as description, images, entry information, and contact details. It also includes an external link to a fashion store website. This project focuses on learning how to structure a real-world webpage using only HTML.

## Features

- Event title and introduction section
- Event descriptions with proper formatting
- Images to represent each event
- External website link
- Contact information section
- Basic webpage structure using HTML

## Technologies Used

- HTML5

No CSS or JavaScript is used in this project. It is created only for practicing HTML fundamentals.
